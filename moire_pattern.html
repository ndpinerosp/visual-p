Moire Pattern Illusion

brightness,Contrast, other

  • 30 Oct

El siguiente programa se realiza la ilusion visual Moiré pattern, como se ve en el articulo de wikipedia Moiré Pattern, Estos son patrones de interferencia visual , que se pueden producir cuando se superpone un patron tramado o de rejillas sobre otro patron similar, y para que aparesca este patron de interferencia los 2 patrones no deben ser identicos, estos pueden ser desplazados, rotados o incluso tener un tono ligeramente diferente. El efecto Moirè consiste en la creación de curvas a partir de la ventana rallada. Cuanto más se reducen las imágenes, más se intensifica este efecto.

Line Moiré.

En esta ilusiom se encuentran dos conjuntos de anillos concentricos (Verde y Morado), el conjunto de anillos de color verde puede ser movido mediante la posicion del mouse, mientras el otro conjunto de anillos se encuentra fijo en la pantalla, al mover el conjunto de color verde superponiendolo al morado se crean bandas oscuras curvas conocidas como "Moirés"

Comandos por Teclado

Comando Operacion
Mueve el disco para generar ilusion

Los dos patrones se superponen, por lo que la luz tiene que pasar ambos patrones (o bloquearse, según sea el caso).television se pueden notar este tipo de patrones cuando alguien usa una camisa con patron de rayas o cuadros muy pequeños, este tipo de patrones de interterencia pueden ser muy agradables a la vista.
